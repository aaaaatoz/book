###关于 《Go 学习笔记》第二版

原本想把《Go 学习笔记》第二版写成一部完整图书，但受到家庭和工作影响，只能暂时放弃。将部分内容做了简单文字校对后，重新上架发布，大概占草稿 1/3 左右。日后将其他内容逐步添加进来。

第二版笔记去掉了标准库内容，改为 **源码剖析**。其一，标准库每次更新都有变化，而且文档也算完备。其次，已经有不少大牛写了标准库部分，我就不掺和了。最后，我会以单章的方式将标准库的某些内容发布出来。
